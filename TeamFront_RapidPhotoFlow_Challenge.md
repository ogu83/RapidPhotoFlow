## 🎯 The Challenge — Build RapidPhotoFlow

A lightweight upload → processing → review workflow.

### Requirements

* Upload multiple photos concurrently
* Simulate async processing
* Display gallery + statuses
* Clean event/workflow log
* Minimal, modern UI

## 🛠 Requirements

### 1️⃣ Simple Modern UI

React recommended. Screens:

* Upload Photos
* Processing Queue
* Review Photos

UI may be generated via:

* Figma → Builder.io
* Figma Make
* Cursor-only

### 2️⃣ Clean Backend Service

Language: any (Java recommended)

Must support:

* Create photo record
* Update status
* List photos

### 3️⃣ Minimal Event-Driven Workflow

Choose:

* Queue
* Polling job
* Processing simulation

### 4️⃣ Zero Handwritten Code

Everything generated by AI.

## 🧠 Tips for Success

* Begin with a 1–2 page design doc
* Use ChatGPT for thinking + architecture
* Use Cursor for scaffolding + wiring
* Keep it simple and clean
